HTML 태그 종류

  1. 줄바꿈 태그 = <br>
  2. 단락 태그 = <p>
  3. 강조 태그 = <strong>
  4. 부모 태그 = <parent>
  5. 자식 태그 = <child>
  6. 리스트 태그 = <li>
  7. 정렬안된 리스트 Unordered List = <ul>
  8. 정렬 리스트 (Ordered List) = <ol>
  9. 제목 및 머리말 = <head>
  10. 본문 = <body>
  11. 문서에 관한 메타데이터 = <meta>
  12. 참조 링크 태그 = <a>
  13. 참조 링크 태그에서 링크 첨부 = <a href="참조링크">
